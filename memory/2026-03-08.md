## EXTERNAL DID Discovery
- btcr2 EXTERNAL type (`x1q5...`) allows custom services in DID documents
- Genesis document hash is encoded in the DID string — tamper-proof
- Tradeoff: need sidecar/registry to serve the document (KEY type is self-resolving)
- Created `create-external.mjs` script in wallet skill
