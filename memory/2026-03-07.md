## btcr2 Breakthrough
- Pivoted from Trustchain (blocked by testnet4 instability) to did:btcr2
- btcr2 supports offline DID creation — no Bitcoin transactions needed
- Built working DID creation, VC signing (bip340-jcs-2025 Data Integrity), verification
- Key API quirks documented: DidBtcr2 (not Didbtcr2), create() returns string, SchnorrMultikey id is fragment-only
- Packages: @did-btcr2/method, @did-btcr2/keypair, @did-btcr2/cryptosuite
- Networks supported: bitcoin, signet, regtest, testnet3, testnet4, mutinynet
